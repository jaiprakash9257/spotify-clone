# Spotify Clone

## Project Documentation
This is a Spotify clone application that allows users to stream music, create playlists, and discover new songs.

## Features
- User Authentication
- Music Streaming
- Playlist Creation
- Song Recommendations

## Setup Instructions

### Prerequisites
- Node.js installed
- MongoDB installed

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/jaiprakash9257/spotify-clone.git
   ```
2. Navigate into the project directory:
   ```bash
   cd spotify-clone
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the MongoDB service.
5. Start the application:
   ```bash
   npm start
   ```

### Usage
- Open your browser and go to `http://localhost:3000` to access the application.
